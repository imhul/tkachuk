# Portfolio React App v2.0.1

### by Tkachuk Zakhar

### [`*** Run in ***`](https://tkachuk-zakhar.netlify.app/)

Based on [`create-react-app-redux`](https://github.com/notrab/create-react-app-redux) and [`Parcel Bundler`](https://parceljs.org)

[![Netlify Status](https://api.netlify.com/api/v1/badges/dc73725a-6b6a-4ae8-bbf5-8d1c78ac3f39/deploy-status)](https://app.netlify.com/projects/tkachuk-zakhar/deploys)

**What's inside?**

-   **Frontend:**
    -   `react` v19
    -   `redux`
    -   `antd`
-   **Backend:**
    -   `netlify`
    -   `@netlify/functions`
    -   `undici`
    -   `reCapcha` proxy server
    -   Telegram bot
    -   Email sending with `nodemailer` + `gmail`
    -   `mongoDB` API
    -   Cookies
    -   JWT with `jsonwebtoken`

**Roadmap**

-   **Frontend:**
    -   `login/logout/register/forgot password/change password` forms
    -   `Redux Persist` [docs](https://github.com/rt2zz/redux-persist)
-   **Backend:**
    -   OAuth
    -   trello API
    -   supabase API
    -   feathersjs + private server

Try the published app [here](https://tkachuk-zakhar.netlify.app/)
